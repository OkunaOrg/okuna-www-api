## Getting started

Clone the repository

```sh
git clone git@github.com:OpenBookOrg/openbook-org-backend.git
```

Install the dependencies
```bash
$ npm install
```

Serve with hot reload at localhost:8080
```bash
$ npm run dev
```

Build for production and launch server
```bash
npm start
```
